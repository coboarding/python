## Menu i nawigacja
- Menu generuje się automatycznie na GitHub Pages (np. Jekyll/MkDocs) i działa także na GitHub.
- Każdy rozdział to osobny plik `.md` w katalogu `docs/`.
- Edytuj `index.md`, by dodać nowy rozdział do menu.
