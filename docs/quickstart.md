# Szybki start

1. Sklonuj repozytorium:
   ```bash
   git clone https://github.com/coboarding/python.git
   cd coBoarding
   ```
2. Uruchom instalację:
   ```bash
   bash install.sh
   ```
3. Włącz system:
   ```bash
   bash run.sh
   ```
4. Wejdź na `http://localhost:8082` w przeglądarce.

Więcej informacji w [Instalacja](install.md).
